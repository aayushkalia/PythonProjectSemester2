## Installation

To use the Voice Assistant, follow these steps:

1. **Install Required Packages:** 
   - Navigate to the project directory and install the required Python packages:
     ```
     pip install -r requirements.txt
     ```

2. **Set up Environment:**
   - Ensure that Python is installed on your system.
   - Verify that your system has a working microphone for speech recognition.

## Usage

1. **Run the Script:**
   - Open a terminal or command prompt.
   - Navigate to the directory containing the script files.
   - Run the script `main.py` by executing:
     ```
     python main.py
     ```

2. **Interacting with the Voice Assistant:**
   - Upon running the script, the voice assistant will greet you and await your command.
   - Speak clearly into your microphone to issue commands.
   - Example commands include:
     - "Open YouTube"
     - "Search Wikipedia for [query]"
     - "Play music"
     - "What is the time?"
     - "Take screenshot"
     - and more.

3. **Exiting the Program:**
   - To exit the voice assistant program, you can say "Go to sleep".
   - Alternatively, you can close the terminal or command prompt window.

## Features

- **Voice Commands:** Interact with the voice assistant using natural language commands.
- **Web Integration:** Perform tasks such as searching the web, accessing Wikipedia, and watching YouTube videos.
- **Utility Functions:** Get weather information, play music, take screenshots, and more.
- **Customizable:** Extend functionality by adding new commands or integrating with other services.

## Troubleshooting

If you encounter any issues while using the Voice Assistant, consider the following troubleshooting steps:

- **Check Microphone:** Ensure your microphone is properly connected and functioning.
- **Internet Connection:** Some commands may require an active internet connection. Verify your connection status.
- **Error Messages:** If the script encounters errors, refer to the error messages displayed in the terminal/command prompt for troubleshooting.
- **Documentation:** Review the README file and documentation for guidance on usage and troubleshooting.

For further assistance, feel free to reach out to the project maintainers or community members for support.
